# C++ Practice Repository

This repository contains a collection of C++ practice programs organized by topic.

## Structure

- 1_Basic and Syntax
- 2_Control Flow
- 3_Function
- 4_Arrays & Strings

## What this repository includes

- Basic syntax and C++ fundamentals
- Control flow concepts
- Function-based problem solving
- Arrays and string manipulation exercises

## How to compile a file

Use the following command to compile a C++ source file:

```bash
g++ filename.cpp -o filename
```

Then run:

```bash
./filename
```

## Notes

This repository is intended for learning and practicing C++ concepts through small programs and exercises.
