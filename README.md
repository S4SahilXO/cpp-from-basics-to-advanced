This repository contains my C++ programming journey from beginner to advanced level. It includes topic-wise coding problems, solutions, examples, practice questions, interview questions, and implementations of important Data Structures and Algorithms.

It is designed for:

- Placement preparation

## ▶️ How to Run

Compile and run a C++ file using:

```bash
g++ filename.cpp -o program
./program
```

For Windows:

```bat
g++ filename.cpp -o program
program.exe
```
